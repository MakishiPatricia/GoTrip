GoTrip

Projeto desenvolvido para a disciplina Tecnologia da Internet.

Protótipo

-Campos obrigatórios
Todos os campos devem ser preenchidos antes de enviar. São eles: Origem, Destino, Data de Ida e Data de Volta. Caso algum esteja vazio, o sistema exibe uma mensagem de erro informando qual campo precisa ser preenchido.

-Data de ida não pode ser no passado
A data de ida deve ser igual ou posterior à data atual. Não faz sentido buscar uma viagem para uma data que já passou.

-Data de volta não pode ser anterior à data de ida
A data de retorno deve ser igual ou posterior à data de ida. Uma viagem não pode terminar antes de começar.
